## Vercel Password Protection

This project is linked to Vercel project `jillian-halley-portfolio`.

To enable Vercel-native password protection:

1. Open Vercel Dashboard and select this project.
2. Go to Settings -> Deployment Protection.
3. In Password Protection, enable the toggle.
4. Choose the deployment scope you want to protect.
5. Set and save your password.

Notes:

- Password Protection is available on Enterprise, or Pro with the Advanced Deployment Protection add-on.
- On Hobby, use Vercel Authentication (Standard Protection) for previews instead.
